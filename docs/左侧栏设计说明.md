# 左侧栏重新设计说明

## 设计参考

参考图片设计，左侧栏重新设计为以下结构：

1. **新对话按钮** - 顶部突出显示的蓝色按钮
2. **快捷功能区** - 包含常用功能的快捷入口
3. **分隔线** - 视觉分隔
4. **历史对话区** - 显示历史对话记录，支持菜单操作

## 实现功能

### 1. 新对话按钮

- **位置**：左侧栏顶部
- **样式**：蓝色渐变背景，圆角按钮
- **功能**：点击创建新会话
- **快捷键提示**：Ctrl+K

### 2. 快捷功能区

包含三个快捷功能：

1. **AI 创作** (Seedance 2.0)
   - 图标：💬
   - 功能：触发AI创作相关功能

2. **云盘**
   - 图标：📁
   - 功能：打开云盘相关功能

3. **更多**
   - 图标：•••
   - 功能：显示更多选项

### 3. 历史对话区

- **标题**：历史对话（大写，灰色）
- **对话项**：包含以下信息：
  - 头像：根据对话内容自动生成
  - 标题：对话的主要标题
  - 预览：对话内容的简要预览
  - 菜单按钮：•••（悬停时显示）

### 4. 菜单功能

点击历史对话项右侧的•••按钮，显示下拉菜单，包含以下选项：

1. **置顶** - 将对话置顶显示
2. **分享** - 分享对话内容
3. **收藏** - 收藏对话
4. **重命名** - 修改对话标题
5. **举报** - 举报不当内容
6. **删除** - 删除对话（红色文字，危险操作）

## 技术实现

### HTML 结构

```html
<div class="sidebar">
    <!-- 新对话按钮 -->
    <button class="new-chat-btn" onclick="createNewSession()">
        <span>💬 新对话</span>
        <span class="badge">Ctrl+K</span>
    </button>

    <!-- 快捷功能 -->
    <div class="shortcut-section">
        <!-- 快捷功能项 -->
    </div>

    <!-- 分隔线 -->
    <div class="divider"></div>

    <!-- 历史对话 -->
    <div class="history-section">
        <div class="history-title">历史对话</div>
        <!-- 历史对话项 -->
    </div>
</div>
```

### CSS 样式

- **新对话按钮**：`new-chat-btn` 类
- **快捷功能项**：`shortcut-item` 类
- **历史对话项**：`history-item` 类
- **下拉菜单**：`dropdown-menu` 类
- **响应式设计**：支持悬停效果和动画

### JavaScript 函数

- `loadHistory(title)` - 加载历史对话
- `toggleMenu(event, title)` - 切换菜单显示
- `pinHistory(title)` - 置顶历史对话
- `shareHistory(title)` - 分享历史对话
- `favoriteHistory(title)` - 收藏历史对话
- `renameHistory(title)` - 重命名历史对话
- `reportHistory(title)` - 举报历史对话
- `deleteHistory(title)` - 删除历史对话

## 交互体验

1. **悬停效果**：
   - 快捷功能项和历史对话项悬停时背景变色
   - 历史对话项的菜单按钮悬停时显示

2. **点击操作**：
   - 点击历史对话项加载对应对话
   - 点击菜单按钮显示下拉菜单
   - 点击菜单选项执行对应操作

3. **键盘快捷键**：
   - Ctrl+K：快速创建新对话

4. **视觉反馈**：
   - 操作后显示系统消息提示
   - 删除操作前显示确认对话框
   - 重命名操作显示输入对话框
